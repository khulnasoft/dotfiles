# KhulnaSoft Dotfiles Documentation

Welcome to the KhulnaSoft dotfiles documentation. This repository provides a comprehensive development environment setup for modern development on macOS and Linux.

## Overview

The KhulnaSoft dotfiles repository is a well-organized setup for:

- **Shell**: Zsh with advanced features (fuzzy menus, Vi mode, forgit)
- **Editor**: Neovim/Vim with extensive plugin ecosystem
- **Terminal Multiplexer**: tmux with smart session management
- **Package Manager**: Homebrew with sync utilities
- **Git**: Enhanced workflow with custom configurations
